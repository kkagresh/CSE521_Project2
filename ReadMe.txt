CSE 521: Project2: To implement a simple virtual memory manager

There is 1 source file in this package i.e.  virtualmem.c. Plus a sample inputRefString.txt file which contains reference string

Compilation Command: gcc virtualmem.c -o virtualmem -lm

Execution Command: ./virtualmem -f 3 -r LFU -i inputRefString.txt

(Note: You can change the above arguments. It is just a sample)
